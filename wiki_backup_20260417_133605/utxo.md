# 

**Summary**: Unspent Transaction Output; the fundamental unit of currency in [[bitcoin-cash](bitcoin-cash.md)](bitcoin-cash.md).

**Sources**: mastering-bitcoin-cash_transactions_4.md

**Last updated**: 2026-04-17

---

A  (Unspent Transaction Output) is an indivisible chunk of currency locked to a specific owner and recognized by the network (source: mastering-bitcoin-cash_transactions_4.md).

### Key Properties
- **No Accounts**: There are no account balances in [[bitcoin-cash](bitcoin-cash.md)](bitcoin-cash.md); instead, balances are derived by summing all s belonging to a user (source: mastering-bitcoin-cash_transactions_4.md).
- **Denomination**: Values are measured in satoshis (1 [BCH](bitcoin-cash.md) = 100,000,000 satoshis) (source: mastering-bitcoin-cash_transactions_4.md).
- **Spending**: s must be consumed in their entirety. Any excess value is returned to the sender as change (source: mastering-bitcoin-cash_transactions_4.md).
