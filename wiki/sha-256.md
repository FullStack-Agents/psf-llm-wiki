# SHA-256

**Summary**: A cryptographic hash function used by [bitcoin-cash](bitcoin-cash.md) for proof-of-work and block hashing.

**Sources**: mastering-bitcoin-cash_chapter-2-how-bicoin-cash-works_9.md

**Last updated**: 2026-04-17

---

SHA-256 (Secure Hash Algorithm 256-bit) is the cornerstone cryptographic hash function used in the [bitcoin-cash](bitcoin-cash.md) network.

### Usage in [BCH](bitcoin-cash.md)
- **Proof-of-Work**: Miners hash the block header repeatedly until the resulting SHA-256 hash is below a specific target value (source: mastering-bitcoin-cash_chapter-2-how-bicoin-cash-works_9.md).
- **Block Hashing**: A double application of SHA-256 is performed on the block header to create the block hash. This double-hashing prevents length extension attacks (source: hash.md). This cryptographic chaining makes the blockchain immutable, as modifying a block requires recreating all subsequent child blocks (source: blockchain.md).
