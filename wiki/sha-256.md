# SHA-256

**Summary**: A cryptographic hash function used by [[bitcoin-cash](bitcoin-cash.md)](bitcoin-cash.md) for proof-of-work and block hashing.

**Sources**: mastering-bitcoin-cash_chapter-2-how-bicoin-cash-works_9.md

**Last updated**: 2026-04-17

---

SHA-256 (Secure Hash Algorithm 256-bit) is the cornerstone cryptographic hash function used in the [[bitcoin-cash](bitcoin-cash.md)](bitcoin-cash.md) network.

### Usage in [BCH](bitcoin-cash.md)
- **Proof-of-Work**: Miners hash the block header repeatedly until the resulting SHA-256 hash is below a specific target value (source: mastering-bitcoin-cash_chapter-2-how-bicoin-cash-works_9.md).
- **Block Integrity**: Because any change to the block data results in a completely different hash, SHA-256 ensures that blocks cannot be tampered with without being detected.
