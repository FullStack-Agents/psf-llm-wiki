# Transaction Script

**Summary**: A decentralized system used for verifying transactions on the Bitcoin Cash network.

**Sources**: mastering-bitcoin-cash_chapter-1-What-is-Bitcoin-Cash-(BCH)_3.md

**Last updated**: 2026-04-16

---

The transaction script is a decentralized transaction verification system and is one of the four fundamental innovations that enable Bitcoin Cash (source: mastering-bitcoin-cash_chapter-1-What-is-Bitcoin-Cash-(BCH)_3.md). 

It provides the cryptographic means to ensure that only the legitimate owner of a digital asset can authorize its transfer, addressing the challenge of authenticity.
