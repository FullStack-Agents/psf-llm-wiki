# Byzantine Generals' Problem

**Summary**: A classic problem in distributed computing regarding how to achieve consensus in a network with potentially unreliable participants.

**Sources**: mastering-bitcoin-cash_chapter-1-What-is-Bitcoin-Cash-([BCH](bitcoin-cash.md))_4.md

**Last updated**: 2026-04-16

---

The Byzantine Generals' Problem describes the challenge of achieving consensus among distributed parties when some participants may be unreliable or malicious.

[Satoshi](utxo.md) Nakamoto's implementation of the Bitcoin network provided a practical solution to this problem by using a [proof-of-work](proof-of-work.md) algorithm (source: mastering-bitcoin-cash_chapter-1-What-is-Bitcoin-Cash-([BCH](bitcoin-cash.md))_4.md). This allows a decentralized network to agree on a single version of the truth (the transaction ledger) without central coordination.

Beyond currency, this solution has applications in:
- Elections
- Lotteries
- Asset registries
- Digital notarization
